# MRK Foods - Premium Luxury Gourmet Ecommerce & B2B Platform

A high-performance, industry-leading web platform built for **MRK Foods Private Limited**, Mumbai. Designed for both HORECA (B2B) hospitality buyers and direct consumers (B2C), this platform showcases luxury imported food brands with a Michelin-chef aesthetic.

## 🚀 Key Features
- **Luxury Theme**: Dark mode #0B0B0B UI with Gold #C6A75A accents.
- **B2B Capabilities**: Wholesale pricing visibility, GST validation, and bulk lead generation.
- **Ecommerce**: Real-time cart management via Zustand and responsive shop with filters.
- **Cinematic Experience**: Ultra-smooth Framer Motion animations and Apple-style interactive elements.
- **SEO Ready**: Dynamic OG tags, Product Schema, and Organization JSON-LD implemented.
- **Tech Stack**: Next.js 14 App Router, TypeScript, TailwindCSS, Framer Motion, Lucide Icons.

## 🛠️ Installation

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Steps
1. Clone the repository:
    git clone https://github.com/your-repo/mrk-foods.git
    cd mrk-foods

2. Install dependencies:
    npm install

3. Run the development server:
    npm run dev

4. Build for production:
    npm run build

## 📁 Project Structure
- `app/`: Next.js 14 App router pages (Home, Shop, Cart, etc.)
- `components/`: Reusable UI modules (Navbar, Hero, ProductCards, Footer)
- `lib/`: Data models, global store (Zustand), and utility functions
- `public/`: High-resolution asset configurations

## 🔍 SEO Strategy
- **Product Schema**: Included for better SERP visibility.
- **Breadcrumb Navigation**: Enhanced for Google site structure.
- **Performance**: Optimized via Next Image for LCP scores.

## 💳 Payments Integration
Uses Razorpay for secure Indian payments (UPI, Cards, Netbanking). Checkout logic is ready for immediate API linking.
